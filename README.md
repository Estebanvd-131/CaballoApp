# EquusApp 🐴📱
EquusApp is an interdisciplinary mobile solution developed through a collaboration between the Faculty of Engineering and the Faculty of Basic Sciences at Universidad Santiago de Cali. This project was created to bridge the technological gap in Veterinary Medicine, providing an interactive educational tool focused on the myology of the Colombian Creole Horse.

## 📖 Project Description
- Learning anatomy traditionally relies on static atlases or cadaveric material. EquusApp transforms this process by integrating:

- Digital Body Painting: Precise identification of muscle groups using real images of a hand-painted Colombian Creole Horse.

- Gamification: Interactive assessment modules (quizzes) that enhance knowledge retention.

- Ethical Impact: Promotes animal welfare by reducing dependence on laboratory specimens, complying with Colombian animal protection laws (Laws 84 of 1989 and 1774 of 2016).

## 🚀 Key Features
- Anatomical Exploration: Hierarchical navigation through anatomical regions and muscle zones.

- Smart Search: Quick localization of specific muscles and their technical data sheets.

- Offline Mode: Data management via JSON files, allowing the app to function perfectly without an internet connection.

- Accessibility: User interface adapted with color modes for colorblind users.

## 🛠️ Tech Stack
- Language: Kotlin (100% Native).

- IDE: Android Studio.

- Architecture: MVVM (Model-View-ViewModel) for clean, modular, and scalable code.

- Data Management: JSON structures for local persistence.
